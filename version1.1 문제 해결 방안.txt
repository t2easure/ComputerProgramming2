문제 해결 방안
저번 version 1.0 파일은 github message로 version 1.0이라고 나와있으며, 파일명이 ‘main_0’, ‘module_a_0’, ‘module_b_0’ 으로 나와있음.
그리고 version 1.1 파일은 message로 version 1.1이라고 나와있으며, 파일명이 ‘main_1’, ‘module_a_1’, module_a_1’ 으로 나와있음.

main 함수에서 사용자가 수능 데이터를 원하는 연도로 선택할 수 있도록 입력을 추가했다. 사용자는 2020, 2021, 2022, 2023 중 하나의 연도를 선택할 수 있으며, 잘못된 입력에 대해서는 오류 메시지를 출력하고 프로그램을 종료하도록 했다. 
사용자가 입력한 연도를 바탕으로 파일 이름을 자동으로 생성하도록 수정했다. 사용자가 2020을 입력하면 '20201231.csv'와 같은 형식의 파일 이름을 생성하여 데이터를 읽어올 수 있도록 했다.
표준점수의 범위가 크므로 1단위씩 끊어서 막대그래프를 생성하면 너무 복잡해지기에, 이를 위해 5점 단위로 그룹화한 열을 추가하고, 해당 그룹을 기준으로 데이터 합계를 계산하여 막대 그래프로 그리도록 했다.
version 1.0에서는 남녀 표준 점수 분포를 선 그래프로 나타냈지만, version 1.1에서는 막대 그래프로 변경했다. 그래프의 제목에 연도를 반영하도록 수정하여, 사용자가 선택한 연도에 따라 그래프 제목이 자동으로 변경되도록 했다. 사용자가 2020을 선택하면 제목에 '2021 수능'이라는 문구가 포함되도록 수정했다.
